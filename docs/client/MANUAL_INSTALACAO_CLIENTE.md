# Smart Tech PDV — Manual de Instalação do Cliente

Este manual é para o cliente final ou técnico de instalação. O cliente não precisa usar Git, npm, cargo, wrangler, terminal ou linha de comando.

## Antes de instalar

1. Confirme que recebeu o instalador oficial `SmartTechPDV-Setup-x.x.x.msi`.
2. Confirme que o arquivo veio do canal oficial de suporte ou atualização.
3. Feche qualquer versão aberta do Smart Tech PDV.
4. Se já existir uma instalação anterior, faça backup pelo próprio sistema antes de atualizar.

## Instalação

1. Clique duas vezes no instalador MSI do Smart Tech PDV.
2. Avance nas telas do instalador até concluir.
3. Abra o Smart Tech PDV pelo atalho criado.
4. Aguarde o carregamento inicial.
5. Ative a licença quando solicitado.
6. Cadastre os dados da empresa.
7. Crie ou ajuste o usuário administrador.
8. Configure a impressora padrão.
9. Faça um teste de impressão.
10. Gere o primeiro backup.

## Primeira abertura

Na primeira abertura, confira:

- nome da empresa;
- telefone e dados básicos;
- usuário administrador;
- licença ativa;
- impressora selecionada;
- teste 58mm, 80mm ou A4/PDF conforme a impressora usada;
- backup inicial criado.

## Atualização por cima

Quando receber uma versão nova, instale por cima da versão anterior. O sistema deve preservar os dados locais do AppData, incluindo clientes, produtos, vendas, OS, financeiro, configurações, licença, `store_id` e `device_id`.

Antes de atualizar, faça backup pelo sistema. Depois de atualizar, abra o Smart Tech PDV e confira se os dados continuam aparecendo.

## O que o cliente final não deve receber

O cliente final deve receber somente o instalador MSI e documentos de orientação. Não envie projeto fonte, arquivos de desenvolvimento, chaves, certificados privados, banco real de outro cliente, logs internos ou pacote de atualização bruto.

## Em caso de problema

1. Não apague a pasta de dados do sistema.
2. Abra Configurações > Sistema > Diagnóstico.
3. Gere ou envie o pacote de suporte, se solicitado.
4. Faça backup antes de qualquer restauração.
5. Chame o suporte técnico.
